Full control from the taskbar

Every feature is a single right-click away. No separate launcher, no bloated UI — just a compact tray menu ready when you need it.

Settings — Opens the settings window to configure duration, opacity, position, and theme.
Disable / Enable Overlay — Toggle the overlay on or off instantly without quitting the app.
Start with Windows — Registers or removes the autostart registry entry.
Exit — Fully unloads Swizzey Switch from memory. No background processes remain.
